# Anagram Check

Responsive GAE app to check whether two phrases are anagrams of each other. Can be viewed at http://anagramcheck-1334.appspot.com/

## Requirements
- [Google App Engine SDK for Python][]

## Tech Stack

- [Jinja2][], [flask][]
- [bootstrap][]
- [Python 2.7][]
